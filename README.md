# Huong Thao Tra

Monorepo fullstack cho website tra sua/tra thao moc:
- Frontend: React + Vite (thu muc client)
- Backend: Node.js + Express + MongoDB (thu muc server)

## Yeu Cau
- Node.js 18+
- npm 9+
- MongoDB local hoac cloud

## Cai Dat
```bash
npm install
npm install --prefix client
npm install --prefix server
```

## Cau Hinh Moi Truong
Tao file `server/.env` tu `server/.env.example` va cap nhat gia tri thuc te.

## Chay Du An
```bash
npm run dev
```

## Chay Rieng Tung Phan
```bash
npm run client
npm run server
```
